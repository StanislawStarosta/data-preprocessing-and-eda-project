# Data Preprocessing, Cleaning and Exploratory Data Analysis (EDA)

## 📌 Project Overview

This project focuses on the process of preparing raw data for analysis through cleaning, transformation, and validation.

The main objective was to ensure data quality and consistency before applying analytical methods.

---

## ⚙️ Methods and Techniques

The following data preprocessing steps were performed:

- merging multiple datasets
- data formatting
- handling missing values (imputation)
- detection and correction of inconsistencies
- exploratory data analysis (EDA)
- descriptive statistics

---

## 📊 Data Preparation Process

### 1. Data Integration
- combining multiple tables into a unified dataset  
- resolving structural inconsistencies  

### 2. Data Cleaning
- handling missing values  
- correcting data types and formats  
- removing or adjusting invalid observations  

### 3. Data Validation
- checking consistency and correctness of data  
- identifying anomalies and potential errors  

### 4. Exploratory Data Analysis (EDA)
- summary statistics  
- distribution analysis  
- basic relationships between variables
